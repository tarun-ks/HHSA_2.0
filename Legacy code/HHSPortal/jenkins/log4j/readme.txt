copy the log4j-1.2.17.jar to weblogic home custom domain lib folder (/opt/apptier/oracle/middleware/user_projects/domains/[domain_nam]/lib/) , for example: for rh7 tst env internal site: /opt/apptier/oracle/middleware/user_projects/domains/HHSAAEDOMAINTST/lib/

for local do the same, needed for both internal site and provider site
C:\Oracle\Middleware\user_projects\domains\[your_domain]\lib
for example:
C:\Oracle\Middleware\user_projects\domains\internal\lib
C:\Oracle\Middleware\user_projects\domains\external\lib
